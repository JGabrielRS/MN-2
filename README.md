# Métodos Numéricos 2

Repositório com os métodos implementados na cadeira de Métodos Numéricos 2 da UFC.

Para compilar e rodar o programa utilize o make:

```bash
make run
```

Na função main, chamamos 4 funções que equivalem as 4 unidades da cadeira, recomendamos que você comente essas chamadas e teste cada uma separadamente.

Adicionalmente, em ./plot.py temos um script utilizado para plotar os resultados do método de diferenças finitas.
